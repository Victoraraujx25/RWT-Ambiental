
rwt-ambiental — Site multi-página (versão inicial)

Arquivos incluídos:
- index.html
- sobre.html
- servicos.html
- galeria.html
- contato.html
- style.css
- script.js
- assets/logo.png

Como testar localmente:
1. Descompacte a pasta.
2. Abra `index.html` no navegador (duplo clique).
3. Navegue pelas páginas: os links apontam para os arquivos locais.

Publicação (opções rápidas):
- GitHub Pages: crie um repositório, envie toda a pasta e ative Pages apontando para a branch `main` (root).
- Netlify / Vercel: arraste a pasta para o painel de deploy ou conecte ao repositório Git.

Editar conteúdo:
- Substitua as imagens em `assets/` (ex: logo.png).
- Para alterar o texto do WhatsApp automático, edite `script.js` (variável preset).
- Para ajustar o mapa, edite o iframe em `contato.html`.

WhatsApp widget:
- Aparece nas páginas: index.html, servicos.html, contato.html.
- Mensagem padrão enviada: "Olá, estou entrando em contato pelo site da RWT Ambiental!"
